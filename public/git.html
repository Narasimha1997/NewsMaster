Power of OpenSource

News Master is completely an OpenSource project, meaning it's not propritery, the source code of the application is available to all, you can clone it, modify it and do whatever you want, Since this is an educational project there is no need to worry about licensing. If you want to be a part of this project then contribute your idea at naraismhaprasannahn@gmail.com or you submit your code at the gitHub link provided.

Developed using React.js

News master is built completely using Facebook's JavaScript client framework called React.js, React.js provides a flexible components based architecture for building the next generation of Web applications, components based architecture means, you can build a web page composed of components, these components can be controlled and act according to changes of state. For more information visit react.js official site. (Image: React.js Barcelona conference 2016).
Visit react.js official

Built with Progressive web technology

Progressive web apps are the next generation of web, With PWAs you can create awesome web experiences, Web apps can behave just like your normal application does, like they can be added to home screen, work offline and support many native features which are not available for a typical web app. For more information on PWA refer to google developers blog.
